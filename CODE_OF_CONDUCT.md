# Code of Conduct

## Our standard

ForgeTools is a tool for the FAF community. Everyone who participates — by opening issues, submitting PRs, or leaving comments — is expected to be respectful and constructive.

Specifically:
- Be helpful, not condescending
- Criticism of code or data is fine; criticism of people is not
- Disagreements about balance values or tool design should stay technical

## Unacceptable behaviour

- Personal attacks, insults, or harassment of any kind
- Spam or off-topic promotion

## Enforcement

Issues or PRs that violate this standard will be closed. Repeat offenders will be blocked from the repository.

This is a small project — let's keep it friendly.
