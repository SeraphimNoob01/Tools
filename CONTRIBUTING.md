# Contributing to ForgeTools

Thanks for wanting to help! ForgeTools is a small community project for FAF players — contributions of any size are welcome.

## Ways to contribute

- **Report a bug** — open an issue using the Bug Report template
- **Suggest a new tool or feature** — open an issue using the Feature Request template
- **Fix something** — fork the repo, make your change, open a pull request

## Pull requests

1. Fork the repository and create a branch from `main`
2. Keep changes focused — one fix or feature per PR
3. Test your changes in a browser before submitting
4. Describe what you changed and why in the PR description

## Data accuracy

ForgeTools pulls live data from etfreeman-db and the FAForever/fa blueprint repo. If you spot a value that looks wrong in-game, please open a Bug Report and include:
- The unit name and stat that looks off
- The value shown in ForgeTools vs the value you expect
- Your source (e.g. in-game, FAF wiki, unit.bp line)

## Code style

- Plain HTML, CSS, and vanilla JavaScript — no build tools, no frameworks
- Keep it readable: descriptive variable names, short comments where the logic isn't obvious
- Avoid external dependencies where possible

## Questions?

Open a Discussion or an issue — there's no wrong question.
